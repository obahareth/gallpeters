Use /api/projection/ to create the map (e.g.: /api/gallpeters).

Selecting countries

Alpha-3 codes can be found here.

1. /api/projection?select=CHN,IND,USA,IDN,PAK

2. /api/projection?select=China,India,USA,IDN,PAK

3. /api/projection?select=China,India,United States of America,Indonesia,Pakistan

4. /api/projection?select=China&select=India

5. /api/projection?select=CHN&select=IND

Customization

1. /api/gallpeters?select=CHN,IND,USA,IDN,PAK&background_color=89CFF0

2. /api/gallpeters?select=CHN,IND,USA,IDN,PAK&stroke=rgb(191, 165, 136)

3. /api/gallpeters?select=CHN,IND,USA,IDN,PAK&selection_color=092

4. /api/gallpeters?select=CHN,IND,USA,IDN,PAK&background_color=89CFF0&stroke=rgb(191, 165, 136)&selection_color=092
